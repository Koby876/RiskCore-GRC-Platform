GETTING STARTED
───────────────
1. Double-click RiskCore.exe
2. Set your access password on first launch
3. Go to Settings → Organisation and enter your details
4. Add Risk → start your risk assessment

YOUR DATA
─────────
All data is stored locally in riskcore.db (created automatically).
Nothing is sent to the internet unless you configure an AI API key.

Keep these files safe — they are YOUR data:
  riskcore.db          All risks, treatments, settings
  riskcore.key         Encryption key (do not delete)
  backups/             Database backups
